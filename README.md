# Ciphers: A Restaurant Search Application

The repository contains all of the necessary code files to run this Restaurant Search application. This was created as a project for the IT490-104, Systems Integration at NJIT. Users will be able to find restaurants around the world. Users can view list of restaurants, restaurantâ€™s menu, and can suggest their favorite dishes to add to the restaurant's menu. Users are also able to rate and review the restaurant. You can see technologies used in the project below.

* Front End: PHP, HTML, CSS (Bootstrap), JavaScript
* Backend: PHP
* Technologies: RabbitMQ, MySQL, GIT
* Restaurants Data Source: Zomato
